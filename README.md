# FastAPI_ML_Project
API for a machine learning application developed using FastAPI.
Проект представляет собой API для модели машинного обучения, предназначенной для классификации изображений кошек и собак. API реализовано с использованием библиотеки FastAPI и обеспечивает возможность получения прогнозов через HTTP-запросы.
